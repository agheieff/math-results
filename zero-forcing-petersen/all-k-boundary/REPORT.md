# Solved exact construction

The parity-dependent column blocks in this package prove a positive all-\(k\) theorem:
\(P(km,k)\) has a half-set of exact internal boundary \(2m\) for every \(k,m\ge3\).

At \(m=k\), this yields an infinite square family with boundary \(2k\), two below the
target \(2k+2\). For \(k\ge6\), these examples also satisfy \(n=k^2\ge6k-1\).

This result concerns a proposed balanced-separator lower-bound route. It neither gives
zero forcing sets of size \(2k\) nor refutes an eventual zero forcing equality.
