# Unique multiset sums

- [Finite abelian groups](finite-abelian/): a size-\(n\)
  unique-multiset-sum family forces group order at least \(2^{n-1}\), sharply.
- [Cyclic groups](cyclic/): in \(\mathbb Z_N\), such a family forces
  \(N\ge2^{n-1}+2\lfloor(n-1)/2\rfloor\).

The linked packages contain the proofs and exact checkers.
