# Tight union-closed families on \([8]\)

Exact certificates verify that no clean tight union-closed family on \([8]\)
containing \(\varnothing\) has between 2 and 23 members.

The certified ranges are split into
[2--15](m02-m15/), [16](m16/), [17](m17/), [18](m18/), [19](m19/),
[20](m20/), [21](m21/), [22](m22/), and [23](m23/).

Each package contains its checker and certificate-replay instructions.
