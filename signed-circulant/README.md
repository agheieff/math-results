# Signed circulants \(C_n(1,2)\)

For every \(n\in\{20,22,24,26,28\}\), exact exhaustion verifies

\[
\min_\sigma \rho(A_\sigma)
=2\sqrt{\cos^2(\pi/n)+\cos^2(2\pi/n)},
\]

with exactly the two twisted switching classes attaining the minimum.

The independently reproducible packages are:
[20](n20/), [22](n22/), [24](n24/), [26](n26/), and [28](n28/).
